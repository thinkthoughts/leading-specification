
# Notebook 00 Contract

1. state the engineering statement;
2. identify the leading constraint;
3. define connected lanes;
4. name engineering objects;
5. define state variables and observations;
6. show the construction sequence;
7. name the next notebook;
8. export a context manifest;
9. generate durable context figures;
10. include a Colab-compatible outputs download cell.
